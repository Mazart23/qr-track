# QR Track - User Guide

## Installation

### Android - First Time
1. Download the `qr-track.apk` file
2. Open the file on your Android device
3. If prompted, enable "Install from Unknown Sources" in Settings
4. Tap "Install"
5. Grant Camera and Location permissions when asked

### Android - Updating
1. Download the new version APK file

## First Time Setup

1. **Select Language**: Choose English or Polish
2. **Set Theme**: Choose Light or Dark mode
3. **Configure Report Interval**: Set how often reports should be submitted (default: 7 days)

## Using the App

### Adding a New Machine

1. Tap the **Scan** button (floating button at bottom right)
2. Point camera at QR code
3. Enter machine name
4. Tap "Add Machine"
5. Location is automatically recorded

### Adding a Report

1. Tap the **Scan** button
2. Scan the machine's QR code
3. Enter report description
4. Tap "Submit Report"

### Viewing Machines

- **Green checkmark**: Report added today ‚úì

### Managing Machines

1. Tap on a machine to view details
2. Tap "Edit" to change the name
3. Tap "Delete" to remove (will also delete all reports)

### Managing Reports

1. Go to "Reports" tab
2. Tap on a report to view details
3. Edit or delete as needed

### Generating QR Codes

1. Go to "New QRs" tab
2. Select count (how many codes)
3. Select size (in millimeters)
4. Tap "Generate"
5. HTML file will be saved - open and print as PDF

## Tips

- **Backup**: Data is stored locally - export/backup regularly
- **QR Codes**: Print QR codes on durable material
- **Permissions**: Camera required for scanning, Location optional
- **Offline**: App works completely offline

## Troubleshooting

### Camera not working
- Grant camera permission in device settings
- Restart the app

### Can't scan QR code
- Ensure good lighting
- Hold camera steady
- Clean camera lens

### Reports not showing
- Check report interval setting
- Verify machine has reports added
